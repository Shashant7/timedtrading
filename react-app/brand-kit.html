Clear Space

Maintain a minimum clear space equal to the height of the logomark (36px) on all sides. Never crowd the logo with other elements.

Minimum Size

Minimum logomark: 24×24px. Minimum full lockup width: 140px. Below these sizes, use the logomark alone.

Animations

Subtle, purposeful motion. Transitions use ease-out with 0.2–0.3s duration. Scroll reveal uses 0.7s. Decorative animations are gentle and infinite.

Guidelines

Do's & Don'ts

Do

  • ✓ Use the dark base (#0b0e11) as the primary background
  • ✓ Keep the green accent for primary CTAs and highlights only
  • ✓ Use Inter for all text — no secondary typefaces
  • ✓ Apply tight letter-spacing (-0.02 to -0.04em) on headings
  • ✓ Use semi-transparent white surfaces for cards/panels
  • ✓ Add glow shadows to primary CTAs for prominence

Don't

  • ✗ Use the logo gradient for large backgrounds
  • ✗ Mix rounded and sharp corners on the same surface
  • ✗ Place green text on green backgrounds — use white or dark
  • ✗ Use more than 3 text color levels in a single component
  • ✗ Stretch or rotate the logo/logomark
  • ✗ Use solid opaque cards — always use semi-transparent

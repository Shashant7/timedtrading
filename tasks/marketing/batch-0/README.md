## Operator checklist (in order)

1. **Skim [`01-personas.md`](./01-personas.md) and [`02-brand-voice-guide.md`](./02-brand-voice-guide.md) first** (≈ 5 min). These are the lenses for everything else. Edit inline if any persona detail is wrong (geo / job title / objection wording) — those edits propagate to future batches.
2. **Review [`03-email-nurture-sequence.md`](./03-email-nurture-sequence.md)**. Confirm the from-address, sender name, and any ESP-specific merge-field syntax (e.g. `{{first_name|"there"}}` may need to be `{{first_name|default:"there"}}` depending on the ESP). Confirm send-time matrix.
3. **Review the 20 anchor X posts** in [`04-x-posts-anchor-20.md`](./04-x-posts-anchor-20.md). Mark the **5 social-proof posts** as the testimonial-collection blocker — they stay in `placeholder` state until consented quotes exist.
4. **Confirm the 3 Reddit drafts** in [`05-reddit-drafts.md`](./05-reddit-drafts.md) match the operator's posting karma history in each sub. If karma is < 100 in any sub, **delay** posting and comment for 2 more weeks instead.
5. **Schedule the long-form YouTube** in [`06-youtube-long-form.md`](./06-youtube-long-form.md). Decide whether to film in one take this week or break it into the on-camera and screen-cap halves over two days.
6. **Confirm the 3 Shorts scripts** in [`07-youtube-shorts.md`](./07-youtube-shorts.md) match a planned pre-market recording slot (these are the lowest-friction asset to ship daily).
7. **Wire the X queue runner** per the contract documented at the bottom of [`x-queue-2026-05.md`](../x-queue-2026-05.md). One-time engineering task — a small Cloudflare Worker cron that reads the YAML, posts via the X API, flips `status`.
8. **Collect consented testimonials** (one short Typeform to top Discord members, offering a free month for consent). Plug returned quotes into:
   - Anchor X posts D1–D5 (file 4),
   - `x-queue-2026-05.md` rows currently marked `status: blocked-pending-quotes`.

Plan for **60–90 minutes** of operator review on Batch 0 (per canonical plan §10).

- **Cold lead outreach.** Same.

## Compliance audit — verified before submission

- [x] No banned phrases anywhere in this batch (`guaranteed`, `risk-free`, `easy money`, `to the moon`, `wagmi`, `diamond hands`, `you'll make X%`, `our users made $Y`, `turn $5K into $50K`, `beat the market`, competitor brand names).
- [x] No personalized buy/sell recommendation in any asset.
- [x] No promise of returns. Every numeric claim is paired with the window AND a disclaimer.
- [x] Disclaimer block is present on every asset that surfaces live or backtest numbers — body (email/post) or alt-text + first reply (X) or last-3-seconds + description first line (Shorts) or pinned comment + description (long-form YouTube).
- [x] Product-surface copy uses third-person (no "you / your"). Off-site copy (X, Reddit, LinkedIn, emails after signup) uses "you / your" with the third-person switch when quoting engine output.
- [x] No screenshots of other brokers' UIs. No named individual users (social-proof posts use `{{user_handle}}` until consented).
